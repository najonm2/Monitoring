## 📧 Email Configuration

### For Development (Current Setup)
Emails are printed to console (no SMTP required).

### For Production
Edit `monitorportal/monitorportal/settings.py`:

```python
EMAIL_HOST = 'smtp.lumen.com'  # Your SMTP server
EMAIL_PORT = 25
EMAIL_USE_TLS = False
DEFAULT_FROM_EMAIL = 'monitorportal@lumen.com'
```

Or set environment variables:
```powershell
$env:EMAIL_HOST = "smtp.lumen.com"
$env:EMAIL_PORT = "25"
$env:DEFAULT_FROM_EMAIL = "monitorportal@lumen.com"
```

### Disable Email Reports

## 🆘 Troubleshooting

### Email Not Sending
- Check SMTP configuration in settings.py
- Run test: `python manage.py send_level3_bi_email --test`
- Contact IT for SMTP relay setup

### Task Not Running
- Verify task exists: `Get-ScheduledTask`
- Check task history in Task Scheduler
- Verify paths in `run_email_report.ps1`
- Re-run setup script

### Database Errors
- Check Oracle connectivity
- Verify database credentials
- Check network access to databases
